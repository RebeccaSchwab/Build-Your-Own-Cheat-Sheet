HTML Forms Cheatsheet

Form tags

Tag Name Description
<form> Form The element used to collect and send information to an external source
<input> Input This element renders a variety of input fields on a webpage such as text fields, checkboxes, buttons, etc.
<label> Label The element that provides identification for a specific <input> based on matching values of the <input>'s id attribute and the <label>'s for attribute.
<select> Select This element is used to create a dropdown list
<datalist> Datalist This element is used to create a dropdown list. Unlike <select>, <datalist> is used with the <input> element to allow user input to be added to the dropdown list
<option> Option This element is used with the <select> and <datalist> elements and is used to create the list of choices
<textarea> Textarea This element is used when creating a text-box for multi-line input, such as a comment section

Input Type Attributes

Attribute Name Description
checkbox Checkbox Creates a checkbox item
number Number Allows the user to enter only numbers and certain special characters inside the field
range Range Creates a slider the user can adjust between a min and max value
text Text Creates a single row input field that users can type text into
radio Radio Button Creates a radio button
submit Submit Creates a 'Submit' button that submits the form when clicked
name Name Creates the "key" of the key/value pair the form needs in order to submit the user input
password Password Creates a single row input field which allows the user to type censored text inside the field

Other Form Attributes

Attribute Name Description
action Action Tells the form to send the data, and specifies where the data is sent.
method Method Tells the form what to do with the data after it's sent
required Required Specifies a field must include a value
min Min Sets a minimum value for the input field. Used with the "number" input type.
max Max Sets a maximum value for the input field. Used with the "number" input type.
minlength Minlength Sets the minimum number of characters that can be entered into the field. Used with the "text" input type.
maxlength Maxlength Sets the maximum number of characters that can be entered into the field. Used with the "text" input type.
pattern Pattern Uses a regular expression to match against the value of the input. Used with the "text" input type.
